### Build and run unit tests 
1. Clone Source Code
    ``` bash
        git clone https://github.com/xuperchain/contract-sdk-cpp.git
    ```
    
1. Build Contract SDK 
    
    ``` bash
        make build
    ```
1. Run unitest

    ``` bash
        make test
    ```
